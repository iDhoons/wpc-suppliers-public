SQL 기준과 검증

이 페이지는 `data/wpc.duckdb`의 `wpc` 뷰만 사용합니다. `wpc`는 `category_label='합성목재'` 납품요구 실거래 subset입니다.

CASE
  WHEN masYn = 'Y' THEN 'MAS'
  WHEN exclcProdctYn = 'Y' THEN '우수조달'
  WHEN cntrctMthdNm = '수의계약' THEN '수의계약'
  ELSE '경쟁/기타'
END AS channel

검증 쿼리: 플래그 조합과 계약방법 조합별 행 수/금액을 확인합니다.

SELECT masYn, exclcProdctYn, cntrctMthdNm, count(*), sum(try_cast(prdctAmt AS BIGINT))
FROM wpc
GROUP BY 1, 2, 3;

생성 기준: 납품요구 실거래 행, 금액 prdctAmt, 단가 prdctUprc. 카탈로그 등록일(rgstDt)은 사용하지 않습니다.
